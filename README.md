# car-price-prediction
This project demonstrates a complete Data Science workflow to predict vehicle prices based on various technical specifications and market attributes. It showcases the application of regression algorithms and data preprocessing techniques.

📈 Project Highlights:
Exploratory Data Analysis (EDA): Investigated feature correlations and distributions to identify the primary drivers of vehicle value.

Data Preprocessing: Implemented StandardScaler for normalization and PolynomialFeatures to capture non-linear relationships.

Predictive Modeling: Built and compared two models:

Multiple Linear Regression: As a baseline for performance.

Ridge Regression: To handle potential multicollinearity and improve generalization through L2 regularization.

Evaluation: Models were assessed using R² Score, Mean Squared Error (MSE), and Residual Analysis to ensure robustness.

🛠 Tech Stack:
Language: Python

Libraries: Pandas, NumPy, Scikit-Learn

Visualization: Matplotlib, Seaborn

Developed as part of a Machine Learning module during my college studies.
